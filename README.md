# tei_hn1_travail
## Chansons québécoises  
Bienvenue dans mon READ ME qui vous permettra de naviguer à travers les différents documents crées dans le cadre du cours d'introduction à l'édition numérique avec TEI.

Cette édition est une édition TEI d'une dixaine d'anciennes chansons québécoises. L'encodage des thèmes avec des balises comme `seg`avec l'attribut `ana` permet de facielement retrouver les thèmes abordés dans les chansons.

Ces chansons proviennent du Fonds de données linguistiques du Québec. [lien](https://fdlq.crifuq.usherbrooke.ca/corpus/cf09e093-ad65-4978-8095-e708c410fd8d/documents/3aff9f04-c105-46d4-9633-93dcaf86d435#texteIntegral)
