# tei_hn1_travail
## Chansons québécoises  
Bienvenue dans mon READ ME qui vous permettra de naviguer à travers les différents documents crées dans le cadre du cours d'introduction à l'édition numérique avec TEI.

Cette édition est une édition TEI d'une dixaine d'anciennes chansons québécoises. L'encodage des thèmes avec des balises comme &lt;seg ana&gt; permet de facielement retrouver les thèmes abordés dans les chansons.

Ces chansons proviennent du Fonds de données linguistiques du Québec.
